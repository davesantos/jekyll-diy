Jekyll DIY
===

DIY Approach to Jekyll. No Themes, No Cruft, No Fuckery.

## Install

```sh
  gem install jekyll bundler
  bundle install
```
## Deploy

```sh
  bundle exec jekyll serve
```

Or, deploy with [BrowerSync](https://www.browsersync.io/) using [Gulp](http://gulpjs.com/)
```sh
  npm install
  gulp
```
